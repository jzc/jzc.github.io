Trivial solution equivalent to a solution existing

February 09, 2020

From Axler's Linear Algebra Done Right, chapter 3.D, exercise 20:

Suppose nnn is a positive integer and Ai,j∈FA_{i,j} \in \mathbb{F}Ai,j​∈F for i,j=1,…,ni, j = 1, \dots, ni,j=1,…,n. Prove that the following are equivalent. (note that in both parts below, the number of equations equals the number of variables):

(a) The trivial solution x1=⋯=xn=0x_1 = \dots = x_n = 0x1​=⋯=xn​=0 is the only solution to the homogenous system of equations

∑k=1nA1,kxk=0⋮∑k=1nAn,kxk=0.\begin{aligned} \sum^n_{k=1} A_{1,k}x_k&=0\\ \vdots\\ \sum^n_{k=1} A_{n,k}x_k&=0. \end{aligned}k=1∑n​A1,k​xk​⋮k=1∑n​An,k​xk​​=0=0.​

(b) For every c1,…,cn∈Fc_1, \dots, c_n \in \mathbb{F}c1​,…,cn​∈F, there exists a solution to the system of equations

∑k=1nA1,kxk=c1⋮∑k=1nAn,kxk=cn.\begin{aligned} \sum^n_{k=1} A_{1,k}x_k&=c_1\\ \vdots\\ \sum^n_{k=1} A_{n,k}x_k&=c_n. \end{aligned}k=1∑n​A1,k​xk​⋮k=1∑n​An,k​xk​​=c1​=cn​.​

The left hand side of both system of equations can be rewritten as the matrix equation AxAxAx, where A∈Fn,nA \in \mathbb{F}^{n,n}A∈Fn,n and x∈Fnx \in \mathbb{F}^nx∈Fn. The matrix AAA induces the linear map TA∈L(Fn)T_A \in \mathcal{L}(\mathbb{F}^n)TA​∈L(Fn) where TA(x)=AxT_A(x) = A xTA​(x)=Ax.

Condition (a) is equivalent to null TA={0}\text{null } T_A = \{0\}null TA​={0}, which is equivalent to TAT_ATA​ is injective. Condition (b) is equivalent to range TA=Fn\text{range } T_A = \mathbb{F^n}range TA​=Fn, which is equivalent to TAT_ATA​ is surjective.

Hence, [Axler 3.69] implies that injectivity and surjectivity are equivalent for linear operators, and as TA∈L(Fn)T_A \in \mathcal{L}(\mathbb{F}^n)TA​∈L(Fn), we have TAT_ATA​ injective iff TAT_ATA​ surjective. Therefore, (a) and (b) are equivalent.
