Regular languages are closed under interleaving

February 08, 2020

Let ⊗\otimes⊗ be an operation recursively defined on strings of the same length such that

ϵ⊗ϵ=ϵxa⊗yb=(x⊗y)ab\begin{aligned} \epsilon \otimes \epsilon &= \epsilon\\ xa \otimes yb &= (x \otimes y)ab \end{aligned}ϵ⊗ϵxa⊗yb​=ϵ=(x⊗y)ab​

so that the action of ⊗\otimes⊗ on two strings "interleaves" the two strings. For example, ab⊗12=a1b2ab \otimes 12 = a1b2ab⊗12=a1b2.

Similarly, for languages A,B⊆Σ∗A, B \subseteq \Sigma^*A,B⊆Σ∗, let

A⊗B={x⊗y:x∈A,y∈B,∣x∣=∣y∣}A \otimes B = \{ x \otimes y : x \in A, y \in B, |x|=|y|\}A⊗B={x⊗y:x∈A,y∈B,∣x∣=∣y∣}

be the set of all string obtained from interleaving a string in A with a string in B. Note that ⊗\otimes⊗ is bijective, so each string in A⊗BA \otimes BA⊗B is uniquely defined by its operands.

Proposition: If AAA and BBB are regular, then A⊗BA \otimes BA⊗B is regular.

Proof. As AAA and BBB are regular, let MA=(QA,Σ,δA,sA,FA)M_A = (Q_A, \Sigma, \delta_A, s_A, F_A)MA​=(QA​,Σ,δA​,sA​,FA​) and MB=(Qb,Σ,δB,sB,FB)M_B = (Q_b, \Sigma, \delta_B, s_B, F_B)MB​=(Qb​,Σ,δB​,sB​,FB​) be DFAs such that L(MA)=AL(M_A)=AL(MA​)=A and L(MB)=BL(M_B)=BL(MB​)=B. Define the DFA

where δN\delta_NδN​ is defined as follows:

δN((qA,qB,L),a)=(δA(qA,a),qB,R)δN((qA,qB,R),a)=(qA,δA(qB,a),L)\begin{aligned} \delta_N((q_A, q_B, L), a) &= (\delta_A(q_A, a), q_B, R)\\ \delta_N((q_A, q_B, R), a) &= (q_A, \delta_A(q_B, a), L) \end{aligned}δN​((qA​,qB​,L),a)δN​((qA​,qB​,R),a)​=(δA​(qA​,a),qB​,R)=(qA​,δA​(qB​,a),L)​

δN^((sA,sB,L),x⊗y)=(δA^(sA,x),δB^(sB,y),L)\hat{\delta_N}((s_A, s_B, L), x \otimes y) = (\hat{\delta_A}(s_A, x), \hat{\delta_B}(s_B, y), L)δN​^​((sA​,sB​,L),x⊗y)=(δA​^​(sA​,x),δB​^​(sB​,y),L)

Basis. We have

δN^((qA,qB,L),ϵ)=(qA,qB,L)=(δA^(qA,ϵ),δB^(qB,ϵ),L),\hat{\delta_N}((q_A,q_B,L), \epsilon) = (q_A,q_B,L) = (\hat{\delta_A}(q_A, \epsilon), \hat{\delta_B}(q_B, \epsilon), L),δN​^​((qA​,qB​,L),ϵ)=(qA​,qB​,L)=(δA​^​(qA​,ϵ),δB​^​(qB​,ϵ),L),

From the lemma, we have

x⊗y∈A⊗B  ⟺  x⊗y∈L(MA)⊗L(MB)  ⟺  x∈A and y∈B  ⟺  δA^(sA,x)∈FA and δB^(sB,y)∈FB  ⟺  (δA^(sA,x),δB^(sB,y),L)∈FA×FB×{L}  ⟺  δN^((sA,sB,L),x⊗y)∈FA×FB×{L}  ⟺  x⊗y∈L(N)\begin{aligned} x \otimes y \in A \otimes B &\iff x \otimes y \in L(M_A) \otimes L(M_B) \\ &\iff x \in A \text{ and } y \in B \\ &\iff \hat{\delta_A}(s_A, x) \in F_A \text{ and } \hat{\delta_B}(s_B, y) \in F_B\\ &\iff (\hat{\delta_A}(s_A, x), \hat{\delta_B}(s_B, y), L) \in F_A \times F_B \times \{L\} \\ &\iff \hat{\delta_N}((s_A, s_B, L), x \otimes y) \in F_A \times F_B \times \{L\} \\ &\iff x \otimes y \in L(N) \end{aligned}x⊗y∈A⊗B​⟺x⊗y∈L(MA​)⊗L(MB​)⟺x∈A and y∈B⟺δA​^​(sA​,x)∈FA​ and δB​^​(sB​,y)∈FB​⟺(δA​^​(sA​,x),δB​^​(sB​,y),L)∈FA​×FB​×{L}⟺δN​^​((sA​,sB​,L),x⊗y)∈FA​×FB​×{L}⟺x⊗y∈L(N)​

Hence, A⊗B=L(N)A \otimes B = L(N)A⊗B=L(N), so A⊗BA \otimes BA⊗B is regular.
